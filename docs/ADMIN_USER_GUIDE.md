## 1. การเข้าสู่ระบบและหน้าแรก

### 1.1 การเข้าสู่ระบบ (Login)
1. เข้าไปที่ URL ของระบบ (เช่น `https://yourdomain.com/admin`)
2. กรอก **Email** และ **Password** ที่ได้รับ
3. กดปุ่ม `Login`
4. หากลืมรหัสผ่าน ให้ติดต่อ Super Admin

### 1.2 หน้า Dashboard (ภาพรวม)
เมื่อเข้าสู่ระบบ จะพบกับหน้า Dashboard สรุปข้อมูลสำคัญประจำวัน:
- **New Users**: จำนวนลูกค้าใหม่ที่เพิ่มเพื่อนวันนี้
- **Active Chats**: จำนวนห้องแชทที่มีการพูดคุย
- **Pending Orders**: ออเดอร์ที่รอการดำเนินการ (รอชำระ/รอส่ง)
- **Today's Sales**: ยอดขายรวมของวันนี้

> 💡 **Tip**: ให้ตรวจสอบกล่อง "Pending Orders" และ "Active Chats" เป็นอันดับแรกทุกเช้า เพื่อไม่ให้พลาดการบริการลูกค้า

## 2. การจัดการแชทลูกค้า (Inbox)

เมนู: `แชทและลูกค้า` > `Inbox`

### 2.1 หน้าจอแชท
ระบบแชทถูกออกแบบมาให้คล้ายกับ LINE Official Account Manager หรือ Facebook Inbox:
- **ด้านซ้าย**: รายชื่อลูกค้า (เรียงตามล่าสุด)
  - 🟢 จุดสีเขียว = ออนไลน์
  - 🔴 Badge สีแดง = ข้อความที่ยังไม่ได้อ่าน
- **ตรงกลาง**: พื้นที่สนทนา
- **ด้านขวา**: ข้อมูลลูกค้า (Profile, Tags, Note, ประวัติการซื้อ)

### 2.2 การตอบแชท
1. คลิกเลือกลูกค้าจากรายชื่อด้านซ้าย
2. พิมพ์ข้อความในช่องด้านล่าง
3. สามารถส่งฟีเจอร์อื่นๆ ได้:
   - 🖼️ **รูปภาพ**: คลิกไอคอนรูปภาพ เพื่อส่งไฟล์
   - 📋 **Quick Reply**: ข้อความที่ใช้บ่อย (เช่น แจ้งเลขบัญชี, ทักทาย)
   - 🛍️ **สินค้า**: ส่งการ์ดสินค้าให้ลูกค้ากดซื้อได้ทันที
   - 🏷️ **Flex Message**: ข้อความรูปแบบสวยงาม

### 2.3 การติด Tag ลูกค้า
ใช้จัดกลุ่มลูกค้าเพื่อให้ง่ายต่อการ Broadcast หรือดูแล:
1. ดูที่แถบด้านขวา หัวข้อ "Tags"
2. พิมพ์ชื่อ Tag (เช่น `VIP`, `แพ้ยากลุ่มA`, `สนใจวิตามิน`) หรือเลือกจากที่มีอยู่
3. ระบบจะบันทึกอัตโนมัติ

### 2.4 การโอนงาน (Assign)
หากต้องการส่งต่อลูกค้าให้เจ้าหน้าที่คนอื่น:
1. คลิกปุ่ม `Assign` ด้านบนขวาของหน้าต่างแชท
2. เลือกชื่อเจ้าหน้าที่ หรือ "เภสัชกร"
3. แชทจะไปแจ้งเตือนที่เจ้าหน้าที่คนนั้น

## 3. การจัดการออเดอร์ (Orders)

เมนู: `ร้านค้า` > `ออเดอร์`

### 3.1 ขั้นตอนการจัดการออเดอร์ (Workflow)

**สถานะออเดอร์ปกติ:**
`Pending` (รอชำระ) → `Waiting Proof` (รอตรวจสลิป) → `Paid` (ชำระแล้ว) → `Shipping` (ส่งแล้ว) → `Completed`

#### กรณี 1: ลูกค้าโอนเงิน (Transfer)
1. ไปที่แท็บ `รอตรวจสลิป` (Waiting Proof)
2. คลิกที่เลขออเดอร์เพื่อดูรายละเอียด
3. ตรวจสอบ **สลิปโอนเงิน** เทียบกับ **ยอดเงิน**
4. หากถูกต้อง กดปุ่ม `✅ อนุมัติ (Approve)`
   - ระบบจะเปลี่ยนสถานะเป็น `Paid`
   - ลูกค้าจะได้รับไลน์แจ้งเตือนว่า "ได้รับโอนเงินแล้ว"
5. หากไม่ถูกต้อง กดปุ่ม `❌ ปฏิเสธ (Reject)` พร้อมระบุเหตุผล

#### กรณี 2: เก็บเงินปลายทาง (COD)
- ออเดอร์จะเข้ามาในสถานะ `Pending` หรือ `Confirmed` (ตั้งค่าได้)
- ให้ตรวจสอบที่อยู่จัดส่ง หากครบถ้วน ให้กดเปลี่ยนสถานะเป็น `Processing` เพื่อเตรียมของ

### 3.2 การจัดส่งสินค้า (Shipping)
เมื่อแพ็คสินค้าเสร็จแล้ว:
1. ไปที่ออเดอร์ที่มีสถานะ `Paid` หรือ `Processing`
2. คลิกแก้ไขออเดอร์ หรือกดปุ่ม `🚚 จัดส่ง`
3. กรอก **Tracking Number** (เลขพัสดุ)
4. เลือกบริษัทขนส่ง (Kerry, Flash, EMS ฯลฯ)
5. กดบันทึก
   - สถานะเปลี่ยนเป็น `Shipping`
   - ลูกค้าได้รับไลน์แจ้งเลขพัสดุพร้อมปุ่มเช็คสถานะทันที

## 4. การจัดการสินค้าและคลัง (Products & Inventory)

เมนู: `ร้านค้า` > `สินค้า`

### 4.1 การเพิ่มสินค้าใหม่
1. กดปุ่ม `+ เพิ่มสินค้า`
2. **ข้อมูลหลัก**:
   - ชื่อสินค้า
   - รหัสสินค้า (SKU) *สำคัญสำหรับการตัดสต็อก*
   - ราคาปกติ และ ราคาโปรโมชั่น (ถ้ามี)
3. **รูปภาพ**: อัพโหลดรูปหลักและรูปประกอบ (แนะนำสี่เหลี่ยมจัตุรัส 1080x1080)
4. **หมวดหมู่**: เลือกหมวดหมู่เพื่อให้ลูกค้าค้นหาง่าย
5. **สต็อก**: ใส่จำนวนสินค้าที่มี
6. กด `บันทึก`

### 4.2 การนำเข้าสินค้าจากระบบอื่น (Sync)
หากร้านเชื่อมต่อกับระบบบัญชีภายนอก (เช่น SeniorSoft, Arincare):
1. ไปที่เมนู `Sync Dashboard`
2. กดปุ่ม `Sync Now` เพื่อดึงข้อมูลสินค้าและสต็อกล่าสุด
3. ตรวจสอบ Log ว่ามีสินค้าอัพเดทกี่รายการ

## 5. ระบบเภสัชกร AI (AI & Triage)

เมนู: `AI & เภสัชกร`

> ⚠️ **สำหรับเภสัชกร**: ส่วนนี้ช่วยคัดกรองเคสและให้ข้อมูลยาเบื้องต้น

### 5.1 หน้า Triage Dashboard
หน้านี้จะแสดงรายการคำถามสุขภาพที่ AI คุยกับลูกค้า:
- **Green Case**: คำถามทั่วไป (ยาวิตามิน, ปวดหัวเล็กน้อย) - AI ตอบเองได้
- **Red Flag**: อาการอันตราย (แน่นหน้าอก, แพ้ยารุนแรง) - **เภสัชกรต้องรีบแทรกแซง**

### 5.2 การแทรกแซง (Intervention)
เมื่อพบเคสที่ AI ตอบไม่ได้ หรือมีความเสี่ยง:
1. ระบบจะแจ้งเตือน Red Flag
2. เภสัชกรกดเข้าไปที่ห้องแชทนั้น
3. พิมพ์ตอบลูกค้าได้ทันที (AI จะหยุดตอบชั่วคราวเมื่อมนุษย์พิมพ์แทรก)
4. ให้คำแนะนำหรือนัดหมาย Video Call

## 6. การจัดการนัดหมาย (Appointments)

เมนู: `นัดหมาย`

### 6.1 ตารางนัดหมาย (Calendar)
- ดูรายชื่อลูกค้าที่จองคิวปรึกษาเภสัชกร หรือรับยา
- สามารถคลิกที่นัดหมายเพื่อดูรายละเอียดอาการเบื้องต้น

### 6.2 การเริ่ม Video Call
1. เมื่อถึงเวลานัด ให้ไปที่หน้า `นัดหมาย`
2. คลิกที่รายการนัด → กดปุ่ม `📹 Start Video Call`
3. ระบบจะส่งลิงก์ห้องประชุมให้ลูกค้าทางไลน์อัตโนมัติ
4. เภสัชกรและลูกค้าสนทนาผ่านหน้าเว็บ (ไม่ต้องลงโปรแกรมเพิ่ม)

## 7. การบรอดแคสต์ (Broadcast)

เมนู: `บรอดแคสต์`

> ใช้สำหรับส่งข้อความประชาสัมพันธ์หาลูกค้าทุกคนหรือเฉพาะกลุ่ม

### 7.1 ขั้นตอนการส่ง
1. **เลือกผู้รับ**:
   - *ทุกคน*: ลูกค้าทั้งหมดที่ยังไม่ Block
   - *Filter by Tag*: เช่น ส่งโปรโมชั่นแพมเพิร์ส เฉพาะกลุ่ม "คุณแม่ลูกอ่อน"
2. **เลือกข้อความ**:
   - พิมพ์ข้อความ Text
   - หรือเลือกรูปภาพโปรโมชั่น
   - หรือเลือก Flex Message (การ์ดสวยงาม)
3. **ตั้งเวลาส่ง**:
   - *Send Now*: ส่งทันที
   - *Schedule*: ตั้งล่วงหน้า (เช่น ส่งพรุ่งนี้ 10:00)
4. กด `ส่งข้อความ`

## 8. สมาชิกและ CRM

เมนู: `สมาชิก`

### 8.1 การจัดการแต้ม (Points)
โดยปกติระบบจะให้แต้มอัตโนมัติเมื่อสั่งซื้อ แต่แอดมินสามารถปรับแก้ได้:
1. ค้นหาชื่อหรือเบอร์โทรสมาชิก
2. กดที่ชื่อเพื่อเข้าหน้า Profile
3. หัวข้อ "Points" กดปุ่ม `Manage`
   - **Give Points**: ให้แต้มเพิ่ม (เช่น ชดเชย, กิจกรรมพิเศษ)
   - **Deduct Points**: หักแต้ม
4. ใส่จำนวนและเหตุผล → กดบันทึก ลูกค้าจะได้รับไลน์แจ้งเตือนแต้มทันที

## ❓ คำถามที่พบบ่อย (Admin FAQ)

**Q: ลูกค้าแจ้งว่าโอนเงินแล้ว แต่สถานะไม่เปลี่ยน?**
A: ตรวจสอบว่าลูกค้ากด "อัปโหลดสลิป" หรือยัง ถ้าส่งสลิปมาในแชทแต่ไม่อัปโหลดในระบบ ระบบจะไม่ทราบ ให้แอดมินกดเปลี่ยนสถานะเป็น "Paid" ให้ลูกค้าได้เลย หรือสอนลูกค้ากดยืนยัน

**Q: อยากหยุด AI ชั่วคราว ทำอย่างไร?**
A: ไปที่เมนู `AI Settings` > ปิดสวิตช์ `Enable AI` ระบบจะเข้าสู่โหมดตอบกลับอัตโนมัติแบบปกติ (Keyword) หรือรอแอดมินตอบ

**Q: สินค้าหมดทำไง?**
A: แก้ไขจำนวนสต็อกสินค้าเป็น 0 ระบบจะแสดงป้าย "สินค้าหมด" และลูกค้าจะกดลงตะกร้าไม่ได้
